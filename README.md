### 快速开始

克隆项目文件:

```bash
git clone https://github.com/modaren/node-chat-room.git
```

进入目录安装依赖:

```bash
#开始前请确保没有安装roadhog、webpack到NPM全局目录, 国内用户推荐yarn或者cnpm
npm i 或者 yarn install
```

开发：

```bash
node index.js #第一次npm run dev时需运行此命令，使开发时编译更快
打开 http://localhost:10086
```
